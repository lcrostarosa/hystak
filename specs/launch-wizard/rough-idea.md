# Launch Wizard — Rough Idea

When I launch a project in hystak I need to be brought to a configuration wizard. Currently it just loads up Claude. In that wizard I should be able to load all the skills, tools, plugins, MCPs — everything. hystak's philosophy is to configure once and use many times.

I should be able to go back and load in MCPs, tools, and skills as I need them mid-session.

Currently the implementation experience is very unpolished right now. We need a logo on boot to be shown, like what happens when you load Claude Code.
